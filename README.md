# 📚 Research Intelligence Agent

An AI-powered Research Intelligence Agent that analyzes research papers using Retrieval-Augmented Generation (RAG) and Large Language Models.

## Features

- 📄 Upload Research Papers (PDF)
- 📝 Research Summary
- 📖 Literature Review
- 💡 Novelty Analysis
- 📚 Citation Gap Detection
- 🔮 Future Trend Prediction
- ❓ Research Q&A
- 📤 Export Results (PDF, DOCX, Markdown)

## Technology Stack

- Python
- FastAPI
- ChromaDB
- Google Gemini API
- HTML
- CSS
- JavaScript

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/Research-Intelligence-Agent.git
cd Research-Intelligence-Agent
pip install -r requirements.txt
```

## Environment Variables

Create a `.env` file.

Example:

```env
GOOGLE_API_KEY=YOUR_API_KEY
```

## Run

```bash
uvicorn app:app --reload
```

Open:

http://127.0.0.1:8000
