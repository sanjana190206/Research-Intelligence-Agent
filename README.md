# 📚 Research Intelligence Agent

An AI-powered Research Intelligence Agent built using FastAPI and Retrieval-Augmented Generation (RAG).

## Features

- 📄 PDF Upload
- 📝 Research Summary
- 📖 Literature Review
- 💡 Novelty Analysis
- 📚 Citation Gap Detection
- 🔮 Future Trend Prediction
- ❓ Research Question Answering
- 📤 Export Reports (PDF, DOCX, Markdown)

## Tech Stack

- Python
- FastAPI
- ChromaDB
- HTML
- CSS
- JavaScript
- Google Gemini API

## Run

```bash
pip install -r requirements.txt
python -m uvicorn app:app --reload
```
